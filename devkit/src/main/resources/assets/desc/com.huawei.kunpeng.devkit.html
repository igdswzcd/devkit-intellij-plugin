Kunpeng Devkit Plugin

IntelliJ provides a series of end-to-end tools for Kunpeng platform application development, porting, performance acceleration, compilation and debugging, and performance optimization.


These tools are plug-and-play, including:

Kunpeng Porting Advisor Plugin

Kunpeng Library Plugin

Kunpeng Compiler Plugin

Kunpeng Hyper Tuner Plugin

Kunpeng Devkit Plugin

  • Application porting and optimization

    • The tools greatly improve software porting and optimization efficiency. They can quickly scan and analyze software to be ported, provide professional code porting suggestions, and provide post-porting performance analysis and acceleration.


  • Application development

    • The tool leverages the basic editing capabilities of IntelliJ to recommend functions optimized by the Kunpeng acceleration library. It also provides Kunpeng compiler and visualized remote debugging.
