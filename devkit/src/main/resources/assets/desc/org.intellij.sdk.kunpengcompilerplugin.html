Kunpeng Compiler Plugin

The Kunpeng Compiler Plugin provides the following functions:

  • One-click deployment
    You can download the plugin from the JetBrains Marketplace and install it online

  • Remote compilation
    One-click installation of the Kunpeng GCC
    One-click installation of the BiSheng Compiler
    One-click installation of the BiSheng JDK
    One-click running of visualized compilation configuration tasks
    Real-time display of compilation and commissioning information
